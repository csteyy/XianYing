# UI 优化方案

## 设计诊断摘要

### AI Slop 信号（需要修正的问题）

- 过度使用渐变和发光效果
- 字体/间距不统一
- 缺乏视觉层级

### 五个优化维度

1. **字体升级**：建立双字体体系（衬线 + 无衬线）
2. **色彩系统**：恢复语义颜色，减少无意义渐变
3. **动效**：收敛过渡动画，保留有意义的微交互
4. **首页重塑**：简化入口，突出核心功能
5. **愉悦细节**：在关键节点添加惊喜感

## CTA 按钮重构

- 从 header 移到底部固定位置
- 返回按钮放大到 44×44px 触控热区
- 删除浮动操作按钮 (FAB)

## 间距与呼吸感

- 统一 padding/margin 系统
- 卡片间距标准化

## 首页新增入口

- 添加「上传文件」入口按钮

## 导航栏迭代

- 尝试 Liquid Glass 风格导航栏
- 后因兼容性回退为标准导航栏
- padding 调整适配安全区

## 语义颜色恢复清单

1. Toast 通知颜色
2. 情绪标注颜色（positive/negative/calm）
3. 图表配色
4. 删除/危险操作红色
5. 录制页 Badge 颜色
6. SoundBars 动画颜色
7. 状态指示器颜色

## 移动端交互优化

- hover → active 状态转换
- 触摸区域最小 44×44px
- 页面转场动画统一
- 滚动行为优化
- 安全区适配
