# 互动场域可视化分析 APP

一个具有科技感和艺术感的互动场域可视化分析移动应用。

## 🔄 用户流程

### 完整录制到可视化流程
```
首页 (Home)
  ↓ 点击"新建记录" 或 FAB按钮
场景设置 (Scene Setup)
  ↓ 填写场景名称 + 添加参与者 → 点击"下一步"
录制准备 (Record Preparation)
  ↓ 检查设备状态 → 确认参与者 → 开始录制
录制进行 (Recording)
  ↓ 实时转录 + 波形可视化 → 停止录制
  ↓ 数据处理中...
数据分析 (Data Analysis)
  ↓ 查看个人数据/发言详情/气氛转折 → 点击"艺术化呈现"
可视化设置 (Visualization)
  ↓ 调整粒子/光束参数 → 点击"生成并完成"
  ↓ 自动保存到作品库
返回首页 (Home) ✓
```

### 快速访问路径
- **查看历史记录**：首页 → 底部导航"记录" → 历史记录列表 → 查看详情
- **浏览作品库**：首页 → 底部导航"作品库" → 筛选和预览作品
- **应用设置**：底部导航"设置" → 录制/可视化/隐私/通知设置

## ✨ 主要特性

### 🎨 全新首页设计
- **增强版标题** - 大尺寸渐变标题 + 发光特效 + 动画效果
- **主次分明布局** - 主要操作大卡片 + 次要功能双列布局
- **视觉层次优化** - 更清晰的信息架构和视觉引导

### 📱 移动APP体验
- **底部导航栏** - 首页、记录、作品库、设置四大入口
- **浮动操作按钮 (FAB)** - 快速创建新记录（带脉冲动画）
- **安全区域适配** - 支持刘海屏等特殊屏幕
- **触摸优化** - 流畅的手势交互
- **毛玻璃效果** - 现代化的视觉设计

### ⚙️ 完整设置系统
- **录制设置** - 音频/视频质量、自动开始等
- **可视化设置** - 默认风格、粒子密度、动画速度
- **隐私与数据** - 转录保存、自动备份、数据保留期限
- **通知设置** - 录制提醒、分析完成通知
- **存储管理** - 查看使用情况、清理缓存
- **关于信息** - 版本信息和许可

### 📚 分离的数据管理
- **历史记录页** - 专注于录制数据、转录文本、时间轴
- **作品库页** - 展示生成的可视化艺术作品、支持筛选

### 🎯 智能导航流程
- **完成即回家** - 可视化生成完成后自动返回首页（带成功提示）
- **中途预览保存** - 可视化过程中可以保存预览，跳转到作品库
- **清晰的引导** - 每个关键步骤都有明确的下一步提示
- **进度追踪** - 场景设置页显示完成进度（场景名称✓ + 参与者✓）

### 🎙️ 录制功能（增强版）

录制页面分为两个核心section：

#### 1️⃣ 实时转录字幕
- 实时显示说话内容
- 自动识别发言人
- 时间戳标记
- 打字机效果的实时文本
- 平滑滚动和动画过渡

#### 2️⃣ 声音波纹可视化
- Canvas实时音频波形
- 60条动态音频柱状图
- 渐变色彩效果（蓝紫色系）
- 发光特效
- 实时音频参数显示（音量、频率、发言人）
